
# Currículo Online - Angelo Malavasi Camillo

## Descrição

Este projeto consiste em um currículo online desenvolvido utilizando HTML, CSS, JavaScript e PHP. O objetivo é apresentar informações acadêmicas, profissionais e pessoais de forma organizada, moderna e responsiva.

Além da exibição de dados do currículo, o sistema utiliza recursos do PHP para gerar informações dinâmicas, como o cálculo automático da idade e a exibição da data e hora de acesso à página.

---

## Tecnologias Utilizadas

### HTML5

Responsável pela estrutura e organização do conteúdo da página.

### CSS3

Utilizado para estilização, responsividade, animações e implementação dos temas claro e escuro.

### JavaScript

Responsável pela interatividade da página, incluindo a troca de tema e o armazenamento da preferência do usuário utilizando Local Storage.

### PHP

Utilizado para geração dinâmica de conteúdo, incluindo:

* Cálculo automático da idade do usuário.
* Exibição da data e hora de acesso.
* Manipulação de variáveis no servidor.

### Font Awesome

Biblioteca utilizada para exibição dos ícones das redes sociais e do seletor de tema.

## Instruções de Execução

### Requisitos

* PHP 8.0 ou superior.
* Servidor local (XAMPP, WAMP ou Laragon).

### Executando o Projeto

1. Copie a pasta do projeto para o diretório do servidor local.

2. Inicie o servidor Apache.

3. Acesse pelo navegador:


http://localhost/curriculo/


4. O currículo será carregado automaticamente.

## Funcionalidades PHP Utilizadas

### Cálculo Automático da Idade

A idade é calculada automaticamente a partir da data de nascimento informada no código.

### Data e Hora de Acesso

O sistema registra e exibe a data e horário em que a página foi acessada.
